# 更新日志 (Changelog)

所有对本项目的修改都将记录在此文件中。

## [1.2.0] - 2026-03-23

升级了依赖分析算法，通过引入 AST 解析机制，显著提升了分析的精准度与对各种导入语法的兼容性。

### 主要功能
- **依赖分析算法升级**：集成 AST 解析支持，替换部分基于正则的依赖提取逻辑，使提取更加准确。
- **带别名导入识别**：支持识别并解析带别名 (Path Alias) 的导入路径。
- **项目层级逻辑调整**：针对用户手动上传的全目录，所有内置文件层级均视为 0，方便统一分析。
- **UI 增强**：项目树中的节点在悬停 (Hover) 时会显示完整名称，优化浏览体验。

### 优化与修复
- **依赖解析缺陷修复**：修复了 TS 中忽略后缀格式的导入语句不被识别的 Bug。
- **多语言支持优化**：解决了部分非 TS/JS 语言在依赖解析过程中的潜在异常。
- **架构清理**：移除前端缓存清理逻辑。

## [1.1.0] - 2026-03-22

此版本重点优化了后端的复杂依赖解析能力及设置项，并引入了后端缓存机制，使解析速度大幅提升。

### 主要功能
- **增加缓存机制**：引入后端文件级别缓存 (`cache.rs`)，大幅提升了对重复项目树或相同环境的解析重放速度。
- **支持可中断解析**：在处理大型项目或漫长依赖树解析时，支持即时中断同步任务，不再占用任务队列。
- **Vue 自动引入支持**：通过优化正侧匹配深度，实现了对 Vue 项目中自动引入型（Auto-import）组件的标签依赖识别。

### 优化与修复
- **工程化拆分**：将核心分析模块 `analyzer.rs` 重构为子目录结构，并按职责分层（`deps.rs`, `resolve.rs`, `ignore.rs`, `constants.rs` 等），提升架构清晰度。
- **配置分类折叠记忆**：精细化设置面板体验，自动记录各配置项分类的折叠/展开状态。
- **输入校验与提示优化**：增加了对非文本文件的过滤警告（拖拽时变红框提示）；为 Number 类输入项添加微调步进（Step）控制。
- **修复压缩参数生效 Bug**：解决了当“压缩触发层级”设为 0 时失效的逻辑问题，使其能够对首层文件生效。
- **引入发版自动化**：新增 `release.js` 脚本及多平台交叉编译辅助，规范了项目的发布流程。

## [1.0.0] - 2026-03-22

这是 CodePulse 的第一个正式版本。主要实现了代码上下文的提取与精简功能，方便将项目代码提供给 LLM 使用。

### 主要功能
- **多语言依赖解析**：支持 TS/JS, Rust, Python 等 10 多种语言的 import/require 递归依赖解析。
- **上下文压缩**：支持在大规模解析时，仅保留类和函数的定义，移除具体实现以节省 Token。
- **结构化输出**：自动生成项目文件树预览，并将代码块格式化为适合 AI 阅读的结构。
- **全新的设置页**：重构了配置项，支持自定义过滤规则、根目录设置和文件类型筛选。

### 优化与修复
- **性能优化**：通过异步任务处理，解决了之前解析复杂依赖时界面会卡死（UI Freeze）的问题。
- **界面调整**：优化了配色方案和滚动条体验，使得在长时间使用时视觉更加舒适。
- **交互改进**：支持文件拖拽解析。
